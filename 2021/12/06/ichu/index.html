竟然翻完了……

包括了Resistance和情报屋的部分。这样所有人物设定应该是翻完了。

这一部分和前面有些part，特别是天天的一些用词，和前面关系解释比较多，比较推荐和前面的一起看。

设定集里还有一篇小短文就再说吧……感觉不会翻……

老样子有问题请轻拍~


【Resistance】

【无铭（無銘）】

椿（Oka，樱花，桜花）

Data

协助Resistance的四人组中的领袖身份。虽然基本是隐秘行动，但需要战斗的时候会使用东之国的刀进行拔刀术，以眼睛无法捕捉的速度将敌人一刀两断。樱花的“巫”之“业”是强化物体。为刀赐予力量之后连岩石都可以斩断。本次前来进行西方调停的内部工作，以将Michael（Yuri）和恶魔拉下台从而可以进行正常邦交为目的。

Secret

真面目其实是本国下任帝王候补中的一人。但是顺位排后。在外交正常化之后便可登上帝王代表的地位。在成为Resistance之后，将作为东之国武器的刀授予了Belial（一诚）和Xaphan（鹰通）。樱花向Belial和Xaphan传授了东之国的剑术。

 

十夜（Gekka，月香，月香）

Data

侍奉樱花（椿）的御三家其中一人，樱花的左右手。自小便来到樱花的家进行侍奉。为了让他（樱花）成为下任帝王甘愿献身。但并不是连自我都消失的程度，而是因为自小看着樱花，相信他是适合掌管国家的人而做出的行动。月香的“巫”之“业”是将自身的姿态变得难以认知。因为掌握这个“业”而擅长谍报活动。并且使用暗器也很强。也很擅长语言，能够毫无困难地使用西方的语言。

Secret

由于训练而善于隐蔽。身体能力也因为训练而出类拔萃，对于毒的耐性很强，连Samael（Lucas）的毒也能使其无效化。成为Resistance之后，指导了Caspiel（双海）使用暗器的“业”。

 

巽（Harukaze，阳风，陽風）

Data

侍奉樱花（椿）的御三家的其中一人。是在东之国进行祭神仪式时以歌祭奉的咏歌一族。但本人比起咏歌更擅长武道。从小时候在剑道上输给樱花以来，对对方一直如亲哥一般仰慕。阳风的“巫”之“业”是使用言灵。面向目标时，他可以使用攻击的言灵来进行攻击。

Secret

出生时便持有强力的“巫”之力，退魔的能力非常杰出。但还做不到自己掌握操控而难以控制。因此，会利用季蝶（葵）的预知来布置“绝对打得中的攻击”。两人聚齐便是攻守完备的存在。还没有掌握西方的语言，说得结结巴巴。

 

葵（Kityo，季蝶，季蝶）

Data

侍奉樱花（椿）的御三家的其中一人。是在东之国进行祭神仪式的舞蹈家一族。在四人中最擅长操纵“巫”之力，能够预知到数秒后的未来。利用这个能力担任着驾驶交通工具的任务。对于樱花，虽然也有家族的因素，但他判断对方是个具有才干的人而跟从侍奉着他。与阳风（巽）的家族都从事技艺，也有作为竞争者的关系。

Secret

“巫”之力比起其他人其实非常少。因为与生俱来的纤细体格而做不到长时间战斗。但他通过超出常人的努力与对美的意识进行锻炼，使自己的“业”达到高手的领域，掌握了即使使用微弱的力量也能战斗的体术和对“巫”的控制力。他的本质其实并不在于才能。

 

【LL(Lost Legion，失った軍勢)】

一诚（Belial，ベリアル）

Data

Resistance的领袖。虽然语气粗暴，但对于将因战祸而荒芜的故乡拯救出来的教会，那份信仰与恩义的感情都是真的。因此曾是现任AC（炼金组）的候补。作为使徒的能力很强，拥有审判的能力。那是能够分辨出可以拟态成人类的强大恶魔的能力。

但是，在某次从任务归来的Belial的路上，他的故乡，一片几乎已经复兴的土地，再次熊熊燃烧了。是因为由于魔女狩猎而变得疑神疑鬼的住民放火而燃烧的。曾拥有相同信念的幼时玩伴三姐妹也在那个时候死去了。因为这个事件，他从教会脱离，成为了Resistance。

堕落之名为Belial，毫无价值之人——

Secret

当他秘密地赶到现场时，发现鲜花被践踏，幼时玩伴的三姐妹中的Edelweiss（Runa）与Mimosa（Momo）的遗体并排躺着。Rose（心）的遗体倒在了教堂里。在去吊慰她之时，她藏在怀里的信滑落了，那是向教会送去了无数次的信。呼唤救援、神父大人似乎已被恶魔附身——阅读了许多信之后，Belial终于理解了真相。自己什么都没有拯救。

 

双海（Caspiel，カスピエル）

Data

在已经消灭的故乡是名士的儿子……似乎是。以前的记忆有些模糊。因为某场战祸遭到了一家失散的不幸。好不容易到达了神学校，遇见了Belial（一诚）与Xaphan（鹰通）。虽然没有干劲，但因为有很高的使徒适性而被半强制地安排了职务。虽然并不情愿，但他还是会对让被派遣到的地方变得悲惨的恶魔感到愤怒。堕落之名为Caspiel，被神囚禁之人——

Secret

在即将完成任务并达到最高序列的时候，由于Rose（心）的泄露，他得知了恶魔与教会的骗局。然后，他记起了教会让他阅读书籍这一事。制造人工使徒的第1号实验成功例。表现出的能力是使役。可以用土做出人偶来同时战斗。对教会充满疑心的Caspiel选择跟随Belial也是理所应当的走向了。

 

鹰通（Xaphan，クザファン）

Data

在已经灭亡的故乡曾是贵族。并且作为使徒的能力也很高，能力是起火。作为应用甚至可以阻止爆炸的才能。了解自己的力量是神明赐予的礼物的同时，他感到了为人们服务的使命感，自己选择入学了神学校。虽然很少见的，家人仍健在，但在成为神之使徒的同时便与家族切断了联系。但作为看重家名的贵族，骄傲地表现自己的样子就如同清廉洁白的骑士一般。

这样的他被Michael承认，也曾有过要成为亲信的传闻，但因为背叛了教会而没有实现。堕落之名为Xaphan，吹响火焰之人——

Secret

知晓了Belial（一诚）的那件事，以及Caspiel（双海）的事件，聪明的他有了对教会的疑问。为了获得对信仰的正确态度，以及证明神明赐予的这个能力的正确性。并且也是为了继续照亮还在被愤怒与绝望包围的同伴。

 

【情报屋（Broker）】

皐月（Jin，金，金）

Data

情报屋双子的一份子。基本以他们自己为优先，多向教会一侧贩卖情报。擅长变装所以不管哪边的国家都可以很好地潜入，同时长于外语。已经死去的父亲与母亲各自生于东方与西方，为了让他们喜欢上两边的国家而教授了两国的语言。双亲死亡之后，两个人在老牧师的家被保护养育。在寻找那时遇见的黑发少年。

Secret

双子虽然在国境被抚养长大，但现实很严酷，（他们）在两边的国家都遭到了歧视。父母被卷进了在边境发生的暴动而死亡。那个时候出现了很多孤儿，双子却没有受到教会的庇护，而是在离学会据点更近的地方被保护在某个老牧师的家。老牧师虽已不在教会登记在册，却是住在边境的人们的心灵归所，并且是很好的商量对象。情报便也自然地流入了双子的耳朵里。那时遇见的黑发少年是双子除对方以外唯一的朋友，但在担保他们身份的老牧师死去的同时失踪了。小时候遇见的黑发少年便是Dark crow（Akio），但金并没有注意到Dark crow的本体。再一次只剩两人的双子，为了活下去而利用在老牧师那里收集的情报，开始购买出售它们。

 

睦月（Yin，银，銀）

Data

情报屋双子的一份子。基本以他们自己为优先，多向教会一侧贩卖情报。而银知道一些金并不知道的事。其中一个便是在双亲死亡的那天，逃离战祸的时候看见了黑发少年的事。那个少年出现在了保护了他们的老人的家前面。怯懦的黑发少年在金银来到之前就在了，但他看见少年在老牧师睡着的时候做了一些事。然后老牧师便再也没有醒来。作为近邻心灵归宿的老牧师死后，周边便充满了猜忌与死亡，双子们也被迫离开。

Secret

银想要知道，那个少年……是不是就是恶魔。这是连对金都不能说的，特别的情报。

【END】


哇我真佩服自己，没想到真的翻完了…………

其实前段时间本来就想做一些外语翻译作为对自己语言能力的练习（说实话并没有什么用因为并没有官方中文翻译让我学习——大部分时候还是要求助moji和deepl——）也是很正好，这个设定真的挺有趣。作为一个待过圈子不多的人，对ichu有一些特别的感情，于是翻了。

比起爱与梦想与偶像我还是比较喜欢这种血海深仇黑暗西幻童话啦！！

第一次做这么大量的翻译，肯定有很多很多各种问题，用词和中文语法都一塌糊涂……但是也竟然没什么人抱怨，肯定是因为我们圈子太冷了哈哈哈哈

那有缘再见！
