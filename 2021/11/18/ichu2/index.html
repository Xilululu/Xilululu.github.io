【IB(Innocent Bless，無垢な神の息吹)】

Noah (Sandalphon，サンダルフォン)

Data

平行方正的少年。可以说是最优秀的存在。他最大的能力是，当认为对方比自己弱小时，便可得到“光之加护”的绝对防御。比起他优秀的使徒资质与高强的战斗力，他作为监督者的能力也毫不逊色。虽然在入学神学校之后他才与儿时玩伴Metatron（星夜）再会，但那时对方已经改变了许多。虽然现在不能像以前那样与Metatron随意交谈，但他希望有一天能与他平等地谈话。在神学院担任赞美诗队的队长。是学生会的实际上的主席。

Secret

与Metatron出身于同一个城市，从小就和他一起加入唱诗班，但有一天Metatron突然消失了。不久之后，整个城市都陷入了战祸，Sandalphon成为了孤儿。是五人小队中神之加护最多的那个，因此虽然本人和IB小队都不知道，但被用作了Metatron的备用品。

 

Leon（Rasiel，ラジエル）：

Data

本来没有加护的资质，但阅读了教会准备的神秘书籍后被赋予了加护。可以说是人工的使徒。Rasiel以外的成功案例还有一个，其他的都成了废人，成为了被恶魔附身的可怜之人被养护院收容。但是，作为补偿而获得了人的身体无法承受的绝对空间掌握能力。那是即使有建筑物和遮蔽物都可以掌握所有在场事物的、可以说是某种全能的力量。但是，如果使用过度会对身体造成伤害，如果连续使用最多只能维持15分钟。经常和Samael（Lucas）搭档。由于神的加护擅长空间掌握所以辅助负责火力的Samael。在神学校担任美化委员。还可以看到他把扫帚当乐器玩的样子。

Secret

作为读书的代价引起了记忆障碍，过去变得模糊了。除了战斗能力的相性之外，还有其他和Samael搭档的理由。因为作为人工使徒被进行了各种各样的实验，对于毒比其他人都具有抗性。虽然面对强力的毒不会死，但是身体会变虚弱。

 

朝阳（Haniel，ハニエル）

Data

由于过强的天使力量影响，与他人目光接触便会魅惑对方，所以倾向低着头。对方是使徒时，虽然因为有加护而可以正常对话，但本人会觉得有些害羞。通过观察敌人，他可以利用“神之光”来揭露对方的原形，暴露弱点。虽然这是非常强大的能力，但因为他本人的力量不足，如果没有一个能击中要害的搭档，就很难奏效。在神学校担当饲养员，为鸽子们起了名字并宠爱着他们。

Secret

在一场席卷城市的火灾中失去了父母。表面上看，这是一场发生在备受内战蹂躏的小国内的争端，但实际上，这是Zadkiel（Rabi）他们试图与恶魔契约而造成的意外。

 

Rabi（Zadkiel，ザドギエル）

Data

曾和挚友一起举行了恶魔的仪式，导致挚友与村民们全都被消灭了。Zadkiel是全村唯一的幸存者。生来便拥有作为使徒的才能，因血液被恶魔讨厌而逃过一劫。以对恶魔有特效的血为媒介进行攻击。为此，他抽取了一定量的自己的血液装入弹夹，作为武器使用。这着实是一种非常损耗身体的战斗方式，但因为尽可能地减少了风险，这方法与Haniel（朝阳）寻找要害的战斗方法非常合拍。言行粗暴，经常被修女·刚训斥。在神学校担服务委员，虽然觉得很麻烦，但也非常认真地对待工作。

Secret

在有着漫长冬天的那一年里，他与挚友举行了一个仪式，条件是恶魔会“结束这个冬天”。但是，恶魔的契约对人类有害无益。那天夜晚，大火吞噬了整个城市，他的挚友也成为了恶魔的活祭。Zadkiel之所以能活下来，唯一的原因就是他的血。他挣扎着，作为幸存者，他不能这么轻易地死去。因此，他选择成为使徒站在前线，承担着靠近死亡的危险角色。虽然他对周围的人声称是因为会被血弄脏而不戴头纱，但真正的原因是，他认为自己没有佩戴的资格。

 

Lucas（Samael，サマエル）

Data

在这个队伍里是最战斗特化型的使徒。对于被称为「赤蛇」的恶魔来说，有产生毒的力量，但是由于会受到精神方面的动摇影响，控制能力仍是今后的课题。另外，神的毒过于强大也会影响普通人，所以不能在人多的地方战斗。和擅长空间掌握的Rasiel（Leon）相性很好，组成了搭档。他曾经居住的村子里发生了狩猎魔女的暴动，失去了青梅竹马的Tigris。年幼的Samael无法阻止大人们的暴行，那些大人们也被火焰吞噬而死。在神学校担任图书委员，有很多绘本藏书。

Secret

Samael从前去世的最重要的人是Kerberos（虎彦），也就是Tigris。仍会做火焰的噩梦。拥有非常强大的力量，但另一方面却最脆弱的。现在还会把Tigris画的遗物画放在吊坠盒里。

 

【AC(Act Ciel，天国の真似事)】

黑羽（Abdiel，アブディエル）

Data

冷静而透彻的、对IB来说是非常好的前辈。也能见到他劝告在战斗时暴走的同伴们那照顾人的一面。虽然曾和LL一样作为使徒消灭恶魔，但LL突然从教会离开，而他从Michael（Yuri）那里听说LL背叛了教会。同样作为使徒，他憎恨着背叛教会的LL。拥有非常高的使徒能力。基本什么能力都很强，不过最擅长的还是近身战。

Secret

在贫民街出生，受到教会的保护之前一直和姐姐街头流浪。因为姐姐被受恶魔操控的人类杀死，非常憎恨恶魔。并且，那时应对这个事件的是和他同期的新人女性使徒，而那位女性使徒因为敌方是人类而犹豫了一瞬间。这使他将对姐姐死亡的怨恨招致到了她身上，主动应募了这个职位，只为了在某天能亲手制裁她。

 

朔空（Raguel，ラグエル）

Data

虽然对人很友好，但永远不知道他到底在想些什么。因Michael的推荐而成为使徒。在听从Michael的命令进行使徒活动的同时，似乎也在单独行动收集着情报。有一个从不离身的少女侧脸的浮雕宝石。热衷于进行对LL的讨伐，擅长中距离的一击脱离【注释】的战斗方法。

Secret

在城市被烧毁的夜晚，一个少女使徒拯救了他的生命。看到她战斗的身姿，他真的认为是“天使降临到了身边”。他的信仰对象不是神明而是那位少女。然而，在某次事件后，她便从第一线退了下来，从那之后也一直受到教会的监视。得知此事后，他就一直梦想着作为异端审问官将她永远关在只有自己的监狱里。虽然这是非常私人的理由，但正因如此，Michael相信他绝对不会背叛自己。

 

【注释】原文是Hit&Away，查了一下似乎是拳击上的术语，接近后打一下马上就跑的方法。不愧是小人（不是）Saku。

 

Baber（Ofanim，オファニム）

Data

基本上一直沉默地工作，因为高大的体格而被小队以外的人们惧怕。但事实上是一个温和的人，容易受动物喜欢。和Haniel（朝阳）一样担任饲养员。有人曾看见过他和鸽子说话，不会真的能理解鸽子的语言吧……？擅长使用高超的使徒能力来发现敌人。虽然和Rasiel（Leon）的能力相似，但他的能力没有那么精确却可以使用更长时间，通用性更高。在使用能力时眼睛会发出更蓝的光芒。

Secret

被教会保护的时候正在边境城镇流浪。起初不能说话，但经过检查后确认他拥有很高的使徒能力，就此入学神学校。脖子上挂着唯一的护身符，是一张写着异国语言的皱巴巴的羊皮纸。紫釉，虽然他在字典里查找不到这个词的读法，但他认为这是神明赐给自己的重要词汇。做这份工作也是因为想要对救助自己的教会报恩，或者说只是因为纯粹的信仰之心吧。

 

【FF(Flame Fate，不死鳥)】

星夜（Metatron，メタトロン）

Data

幼年时期与恰好在视察某个城市的教皇候选人Michael（Yuri）偶然相遇了。他的使徒能力是如此强大，以至于被当场召为使徒。除了银枪，他还能够使用强大的点燃能力在自己身上缠绕火焰之翼，形成绝对的保护墙。并且他也能够运用恶魔的武器在脚上制造木桩，将敌人穿刺，是一个拥有可攻可守的强大能力的人。非常崇拜Michael。

Secret

和Sandalphon（Noah）曾经是挚友，但现在已经想不起来了。当听到Sandalphon从教堂传出的歌声时，会停下聆听，却不知为何自己会觉得怀念。

 

奏多（Kusiel，クシエル）

Data

被遗弃在神学校门口的孤儿。因为作为使徒的能力出类拔萃而被纯粹地培养长大。拥有像天使般纯真的笑容与比起年纪来稍低一些的感性，从来都不放下手里的玩偶。非常喜欢Metatron（星夜）与Kamael（晃），并希望三人能永远这样一起守护世界。因为在能够质疑之前便被灌输了教义，无法容忍那些反叛教义的人。同时，在涉及到教义以外的事情时又极其软弱。情绪高涨的时候会发出飞散的红色火星。会将持有的玩偶变成红色鞭子进行战斗。

Secret

虽然与Mimosa（Momo）是实际上的兄妹，但因为Kusiel和Mimosa都不认识对方而持续着战斗。虽然没有任何人教他，但他记得仅此一首的、圣歌之外的摇篮曲，会在独处的时候悄悄地哼唱。

 

晃（Kamael，カマエル）

Data

因为母亲在魔女狩猎中被判为魔女，他自己也被认为是魔女之子而遭到迫害，而经历这些事也让他有了“这个世界上的弱者在没有领导者的情况下就会重复这样的错误”这样扭曲的认识。因为Kamael有着作为使徒的能力，而且是一个强大的使徒。而人们必须得到指导。拥有神明赐予的力量与恶魔传授的知识，身披红色铠甲，手握鲜红的剑，他会将所有反对Michael的人砍倒。

Secret

知道Michael与恶魔联手这一事。但是，他认为恶魔之力也是神的力量的一部分，并对基于教会的政治也抱持肯定态度。

 

【教会（Church）】

Yuri（Michael，ミカエル）

Data

教会的领导者。随时准备帮助那些受伤的人，随时对需要帮助的人伸出援手，脸上总是带着温和的微笑，一个如神明一般的存在。赋予了在他身边忠实尽责的FF特殊的力量，并经常将他们与Kizaki神父留在身边。干预政治，并在实际上掌握着教会。严格控制着与东之国的邦交关系，以防止其思想进入本国。

Secret

虽然在出生时便已死亡，但他的母亲与恶魔契约，使他复活了。而他又从魔鬼那里得知了自己的母亲是真正的魔女。不知道谁才是真正魔女的人们持续着战争，怀疑并杀害被误认是魔女的无辜之人。知晓这些人类罪恶的Michael形成了恶也是善的扭曲的伦理观。有着这样善良的邪恶的Michael，为了合理地推进一切，再次召唤了出生时便有过联系的Laplace’s demon（拉普拉斯的恶魔，子规）与其联手操控了国家。利用着人心，有计划地构筑着魔女狩猎与战争来控制一切。

最初是两个大国与几个小国进行着战争和小范围冲突，但现在以教会的名义，他们在名义上进行着合作。实际上的确拥有着很高的才能。能力是精神感应。有许多种使用方法，包括让对方信任自己。发现了在刚懂事时遇见的Metatron的能力，当场对他的父母进行了洗脑来使他们同意带他走。虽然在普通人看来就是个疯子，但因为他的作为，西之国正在从漫长而疲惫的战争中缓缓恢复。
