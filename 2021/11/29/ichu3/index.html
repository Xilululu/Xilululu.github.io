【学会】

Eva（Hei xiao，黒梟）

Data

名唤黑枭的魅惑人心的男人。谁也不知道他的年龄、出身等一切与黑枭有关的事情。是推进学会所设立的“RE:BERSERK（狂战士再生系统）”计划的中心人物。最初由这个计划所唤醒的便是黑猫（澪）与黑兔（蛮），而这两人对于黑枭来说也是像自己的孩子一样重要的存在。可是，黑猫曾被认作是魔女而和黑兔一起成为了无法言语的尸骸。自己第一次如此珍视的东西被夺走，愤怒而绝望的黑枭选择使用了已被他们一族封存的秘术。那便是叫做“RE:BERSERK（狂战士再生系统）”的死者复活之术。为了实现强力的死者复生之术，与恶魔Dark crow（Akio）和Laplace’s demon（拉普拉斯的恶魔，子规）签订了契约，领导着他们。

Secret

曾经是因为魔女狩猎而失势的王族血脉的一人。一族的人们都被杀害，而他独自一人活了下来。虽然隐藏着身份在山间的一个小小的家居住，但因为无法对被抛弃在森林的孩子置之不理，便抚养了他们。而且即使拥有遗留给王族的秘术之类的财产，但觉得只要可以安静的生活便没有问题，一直如此生活着。但是某天，作为让人民发泄愤怒的借口，住在森林里的黑枭一行被盯上了。因为“Michael大人的预言”，“魔女偷偷地住在这里”。黑枭回来时，黑猫与黑兔都已经变成了无法说话的尸骸。过去已经抛弃的本名是紫釉。

 

澪（Hei mao，黒猫）

Data

虽然总是不高兴的样子，但只要黑枭（Eva）下达命令便会像人偶一样听从。自复生后便得到了诅咒的能力。使用藏在长长袖子下的暗器战斗。从黑猫那里受到的伤像冻伤伤痕一样残留着。将黑枭视作主人，颔首低头的样子让黑枭非常痛心。

Secret

在森林里被抛弃的婴儿。曾在黑枭的养育下茁壮成长。过去一度要被黑枭送还到城市里，却和黑兔（蛮）一起紧紧拉着黑枭的手，立下了绝对不会分开的誓言。生前和黑兔是总吵架的好朋友。会带头帮忙制作黑枭每个月一次去城市售卖的膏药，曾是个知识丰富的好孩子。

 

蛮（Hei tu，黒兎）

Data

因为死者复生之术而复生后失去了本来温柔的性格，像小丑一样嘻嘻地笑着。并不知道现在还是否拥有意识。获得了诅咒的能力，因为肉体比活着的时候强化了而擅长近身战。碰触黑兔后会留下像火伤一样的伤害。将黑枭（Eva）视作主人，颔首低头的样子让黑枭非常痛心。

Secret

在森林里被抛弃的婴儿。不管遭受的境遇如何，总是用天真烂漫的性格支持着黑枭与黑猫（澪）。因为和黑猫一样的理由而被杀害。受到黑枭的托付，为了照顾有些感冒的黑猫而留在家里的时候，收到了村人的袭击而被杀害。

 

虎彦（Kerberos，ケルベロス）

Data

由黑枭的计划而被复生的其中一人。参加学会的袭击时会用尖锐的钩爪劈开牺牲者。拥有高超的战斗力，虽是狂战士却也拥有领导的品格。还残留着语言能力，但似乎已经没有过去的记忆了。

Secret

住在森林里的怪人画家。画出的画作虽被称为“异想天开”，但因为会画出像是被附身了一样具有魅力的画作，不知不觉被周围的住民们传作和恶魔在私通的样子。因为和被称作恶魔之兽的野兽的狼亲近的样子被村子的住民看见，而受到袭击死亡。原来曾和Samael（Lucas）是儿时玩伴。

 

恭介（Pure boy，無垢なる物）

Data

可以经常看见他和Kerberos（虎彦）一起行动的场面。非常健忘，而且非常小孩子气。使用和他小小的身躯不相符的大柴刀进行攻击，非常讨厌Kerberos受到攻击，会挺身而出来保护他。

Secret

被抛弃在森林的孩子。为了让狼吃掉他而被包裹上了红色的布。被老猎人捡到抚养。虽然在猎人死去后便一个人居住，但和移住到森林里的画家非常合得来，交到了有生以来的第一个朋友。在某天发现了失去了父母的小狼，就那么开始饲养它。狼在成长的同时变得很亲近人，于是将它介绍给了画家。可在某天去画家家里玩的时候，与画家和狼一起被村子的住民杀害了。在将死之时，他许愿狼和画家都可以死而复生。因那个愿望被扭曲地实现而复活的画家便是Kerberos，作为地狱的走狗而重复着杀戮。

 

Akio（Dark crow，ダーククロウ）

Data

虽然总是软弱的态度，嘴角却又一直挂着愉悦的微笑。在漆黑的夜晚里，隐藏身姿无声地用钩爪切割。像是可以穿越阴影一般，总被目击在各地神出鬼没的样子。

Secret

本体是八咫鸟，鸟神。与Laplace‘s demon（子规）相识而邪神化了。通过总跟着黑枭（Eva）的猫头鹰之间的鸟类的联系而被黑枭所召唤出来。被黑枭召唤之前一直将离国界很近的森林作为根据地。盯上了住得近的老牧师，企图使他堕落，却失败并夺走了他的生命。

 

子规（Laplace’s demo，ラプラスの悪魔）

Data

穿着白衣、像是医生的青年。脸上总是带着温和的微笑。但只要看到他赤红的眼瞳就能明白他并没有理性了吧。一旦被下达了命令便会忠诚地、精心地进行杀戮。

Secret

虽然看起来像是被黑枭（Eva）叫出来的，但实际上是被Michael（Yuri）所唤出的恶魔。在Dark crow（Akio）被黑枭所召唤的时候装做被一起召唤出来的而已。与Michael联手，散播政变与魔女狩猎的谣言，将憎恶作为诱饵来规整这个世界。穿着白色的衣服是为了作为拉普拉斯的恶魔来表达对教会的支持。

 

辉（Blizzard queen，ブリザートクイーン）

Data

缠绕着血染外套的、外表美丽的青年。他一直在生气。一次又一次地边发怒边进行杀戮。那份愤怒会将周围都冻结，染上白色。这个诅咒的范围之广，以至于大部分使徒都无法接近他。

Secret

因为恶魔的策略而被消灭的小国的王子。在持续了漫长冬天的这一年里，由国民传谣“是因为女王成为了魔女”而起，最终政变发生而遭到了迫害。女王被处刑之后，王子无论如何都想让挚友逃走而披上了死去母亲的外套，作为诱饵却被子弹击中倒下了。因为憎恶，他被诅咒的红色玻璃鞋迷住了。为了红色玻璃鞋，成为了倾注生命之血的怪物。

 

乐（White beard，白髭）

Data

被披风紧紧遮住眼睛的男人。可以从间隙里看到他浓密的银发。他一直在微笑。 在被Blizzard queen（辉）冰冻住的雪白世界里，仿佛是为了护送他一般。用烛台的火焰将那些不在女王面前垂下头的家伙们燃烧殆尽。

Secret

小国丞相的儿子，被人们认为和恶魔私通而遭到肃清。虽然处在将来要辅佐王子的位置上，但两人从小就非常亲密，而虽然准备守护王子到最后，途中却和王子失散了。在他因受到枪击而满身疮痍的时候，再次见到的王子却已死去。在将死之时与恶魔签约。王子再次苏醒之时却已经变成了怪物。他祈祷自己也能变成同样的东西，恶魔却没听到这个愿望。维持着理智的白须，沉浸在了疯狂之中。

 

心（Rose，ローズ）

Data

由黑枭（Eva）复活的死者。生前与Edelweiss（Runa）和Mimosa（Momo）像是姐妹一般被抚养长大。比起Edelweiss和Mimosa残留了更多人类的理性。同情为了重要的人而持续研究的黑枭。拥有精神感应（精神的治愈）的能力，在Edelweiss与Mimosa失去理性的时候非常有用。对于其他的死者们也是有效的能力。

Secret

与Edelweiss和Mimosa像是三姐妹一般被抚养。小时候，村子因魔女狩猎被烧毁，而她被孤儿院收留。长大了之后三人都在教会作为修女工作。与Belial（一诚）是幼时玩伴，也交换了许多次书信。并且热心学习，试图比其他人更能帮上神父。但这却成为了错误，他交出的一切——村庄的动向、对魔女狩猎的不安、对教会本部的求援——都被压制了。在教堂没有人的时候，（Rose）怀疑了讲话的神父，又发现了藏在他桌子里的自己的信，虽然想尽办法求救，但为时已晚。最后对Belial留下了关于一切的告发文，作为魔女被杀害了。

 

Runa（Edelweiss，エーデルワイス）

Data

睁开眼睛的时候已经成为了拥有不会跳动的心脏与青色皮肤的尸体。生前是静静守望人的好姐姐，但现在已经成为了为了守护两个人而不择手段的极端的性格。那两人要是被谁伤害了会进入半狂乱状态。可以使植物活性化来操纵它们，但对火很弱。而且看见火便会回忆起生前的记忆陷入恐惧之中。

Secret

是孤儿院院长的女儿。与Rose（心）和Mimosa（Momo）一样被夺去了性命。成为死者之后，依靠残存的少许理性，偶尔会觉得自己并不是这样的，却因为记忆朦朦胧胧而并不明白。虽然对此有强烈的不安，但会对自己说只要有那两人在就没关系。因为如此，对黑枭有着信仰一般的感觉。虽然神明并不存在，但拯救了他们的怪物却无比亲切啊。

 

Momo（Mimosa，ミモザ）

Data

现在已经基本没有了自我，对着玩偶唱摇篮曲，和姐妹们快乐地生活。不管活着还是死了，她都觉得和姐妹在一起是最幸福的事。就算照耀着的不是太阳而是月亮，睡下的不是床铺而是棺材。Mimosa会使用能力制造玩伴。即使她认为的人偶是不会说话的尸体。

Secret

被遗弃在神学院前的孤儿。虽然与Kusiel（奏多）是双胞胎兄妹，但因为没有作为使徒的能力，被送去了孤儿院。虽然生活算不上富裕，但她和温柔的孤儿院孩子们像姐妹一样被幸福的抚养长大。直到魔女狩猎再次袭击村庄的那一天。
