王耀与幽灵

发表于 2014-08-05 更新于 2022-01-22 分类于 同人文 Disqus：
2.4k 8 分钟

没想到我也会写这种已经被写烂了的梗【说的好像以前也没写过大众梗一样】
练手练习，试试写原作漫画的气氛
原作是看的《雪女与幽灵》

在承认自己独居之后，王耀的生活变得异常空旷起来。
像是自己曾经生活在漫画世界里，身边许多许多对话都是一个个对话框，还都是大段大段的长对话，挤满了身边可以触碰到的任何一个空隙，而他只能被那些话语夹着走。拥挤却又充盈的日子。
而现在他把公文包放在餐桌上，面前空空荡荡的也没有人为他准备晚餐，其实也不是晚餐的时间了，他应该喝杯咖啡然后继续工作。
手指舒展一通，右边好像触及了什么温润的硬物，透着香味和热度。
一杯刚刚冲泡的热咖啡。
他的眼睛半闭起，“又是咖啡啊。”
「我才没那么会泡咖啡呢。」
「不止每天一壶啦……都是为了你。」
湿润的咖啡香气，并不提神，反而温暖得让人想依靠。
他突然睁开眼，拿起咖啡杯，把热饮哗啦哗啦地喂给了下水道。
瓷杯被扔在水槽里，他没有管杯口是否已经裂开，把这种事情抛在身后，就这么冲上二楼。楼梯间被踩得咯吱咯吱也没有抗议，还让他稳稳地站在二楼拐角的小阳台上。阳台很小，一小架延伸出去的花台，摆着五六个刚萌芽的小幼苗，低头怀抱自己的样子也真像绿色的天使，他却一推手，把他们连同花瓶一齐推下支撑架。那是摔下两层粉骨碎身的捷径。
「我们的花园里一定会开满花。」

床铺是冰凉的，毕竟也很久没有人让它像被点燃一样温暖过。王耀为自己掖掖被子，玻璃因为内外的气压差被迫推出咯吱声，再加上那么几点打鼓一般的雨点，他听出来下雨了。
即使下雨了月光还是艰难地透出一些，投射在屋内玻璃可以笼罩的地方。
有人怕雨，有人喜欢光。
具备这两点的人踩在月光里一步步拖行，王耀听见了脚步的拖缓声。张开眼，有人穿着一双拖鞋出现在月光可见的范围里，没有光亮的地方又是一片空荡。他上身照不到月亮，于是只有腿与拖鞋被王耀的视线所捕捉到，如同恐怖片里的场景。
王耀一下子慌了神，发现它不能存在于没有光的环境里，赶紧爬起来把窗帘都拉上。
一片眼睛不能适应的黑暗，他大口喘气地蹲下，努力使自己平静下来，再仔细竖起耳朵。没有拖鞋，没有他人的喘息。
没有人。

他在早上出门前发现了放在茶几上的折纸，一支向日葵，有谁帮他捡回来的，纸都还没褪色。
他突然觉得不能再这样了。
打了电话请假，他把外套脱了就开始动手。从卧室里抱出一大堆盒子，里面是各种相片还有其他什么。经过卧室连带着的浴室，发现里面蒸汽满天，竟然已经贴心地放好了一池热水。
「今天休息？先泡个澡放松一下吧。」
不会有人看到他的身体整个都抖动起来，非恐惧非害怕，他只是愤怒地颤抖，在嘴唇都要咬破的边缘将手中的纸盒都往浴池里用力扔去。水面立刻漂浮着许多合影，高个子一些的人不管是边缘还是面孔都模糊不清。
那一扔也似乎耗尽了王耀的力气。他无力地蹲坐，头仰起，让身边烦躁的蒸汽托起自己眼角的液体。
他用压制已久的悲伤掏空胸口的最后一些空气来嘶吼：“你够了吧？你够了吧！我讨厌你！你快给我走！”
这一下似乎有效果，蒸汽停止上升，它们下落到原点，在浴室里的任何地方凝出水珠。一池热水也变得冰凉。不知是不是因为这还是冬天，它们在这温度里似乎马上就要结冰。
王耀却呆滞了一会，哆哆嗦嗦地站起后把水给放了，捞出已经湿哒哒的相片们，把它们都摆在院子里。
还有拾回昨天扔碎的花瓶，还有拿出不合自己大小的衣服，还有大号的碗筷，还有一支钢笔，还有，还有……
还有一支向日葵，一束里的一支。

“他这病情没有好转呐。”
“你是时候放开我的哥哥了。”
“你待在这里明明什么都做不到！”
“不，小耀，你还能为我折一束不会枯的向日葵。”
“让我们缅怀逝者，记住他在这世间所做过的一切——”
“我把这束花放在他的棺材里，请问可以吗？”
“大哥哥，这只花从棺盖里滑出来了哦。”

没错啊，那支向日葵是他自己捡回来的。

他颤抖着手把东西都架了起来，那支纸花放在了最上面。
冬天的风毫不留情地阻止他，两次点燃的火柴都在抖动着的手里熄灭。他不甘心地抓起一摞报纸，卷在手里，点燃了这根引火棒，想走上前点燃让那幽灵寄宿的东西。
可这一次的阻止更加不留情面，风把报纸吹弯了，本来指向前方的火焰突然朝向了自己，还没来得及甩开，就已经有火星点上他的袖口。
“糟糕……！”冬天的衣服本来就厚实又易燃，火势一下子蔓延开，眼看着火就要沿着手臂一路烧上，幸好比较厚，手臂暂时只感觉到灼热却还没疼痛。可是这周围没有水，王耀慌张地甩手臂，火势一点小下去的痕迹也没有。
这样下去会变成自焚吗？其实这样也好。这样，就能再一次真正的见到那个人吧——
“伊万。”他今天第二次哭了出来，和上一次不同的是，说着真心的话。
就在这时，一桶水及时地冲了下来，浇了他满头满脸，却也把手臂上的火焰扑灭了。水落完后水桶就掉在一遍，仿佛刚刚有人举起了桶来浇水。
又似乎有人把王耀抱住，半透明的手轻轻摸着他湿掉的头发，一下一下，在平息他的哭泣。
王耀埋在这团久违又怀念的气息里，却哭得更厉害了：“你怎么……又……出来……快给我……走……别……回来了……”
「好的，小耀，我马上走。你也别哭啦。」
“他”引着王耀站起来，又放开他，站到那些属于自己的东西前。物品周围引起细小的火焰，好像蜡烛。
「我走啦，小耀。」
蜡烛们失去控制，迅速往上燃烧。火焰点燃亡灵的衣角，也把“他”的脸庞照亮，最后一次清晰。
面对这一切王耀只是抽噎地捂住自己的眼睛，让自己不再去看。反正不管如何，他相信“他”最后也是笑着的。
再睁开眼睛，向日葵烧得只剩一瓣。
他再眨眨红肿的眼睛，那一瓣也变成了灰烬，被风吹起，也散开在冬天灰色的天空中。

回房间时头异常的疼痛，王耀觉得应该是感冒了，毕竟在冬天里被浇了冷水，又在冷风里吹了大半天。
他倒在床上，任性地抓过枕头抱着，张开眼，发现床头放了一杯水和感冒药片，似乎是很早以前就摆在那儿的。
“真是混蛋。”他不去理它，又闭上眼，“要走也不带我一起走……”
【END】
