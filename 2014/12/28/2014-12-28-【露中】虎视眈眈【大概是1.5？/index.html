虎视眈眈【大概是1.5？

发表于 2014-12-28 更新于 2022-01-22 分类于 同人文 Disqus：
本文字数： 1.6k 阅读时长 ≈ 3 分钟

有声音唤起了伊万。

“擦。”火机摩擦的声音。

纸张折叠，什么东西快速燃烧又快速熄灭，然后火机又摩擦，然后金属合起。

“咻——”长时间的吐息，强行呼吸，随之而来的味道让伊万闭着眼睛的时候仍旧皱起眉。

是烟味。

他对这东西没有研究，不过大概也猜到了七八分。他悄悄张开眼，浅蓝混黑的幕景布外，窗户制造的纯黑环境里，只能依据大小判断远近，靠前的位置，有一颗火星。

它忽明忽暗，随着点烟人的呼吸变化。伊万盯着它，吐息，吐息，吐——息。

他和他的呼吸逐渐同步于一体。

忽有触感爬上脊背，两根手指模仿走路的动作，顺着他的脊柱向上攀登。擦过的皮肤像水面一样激出涟漪，把伊万昨夜残存的欢愉一丝丝荡了出来。

他醒了。

伊万背过身。向上的两只手指失去了道路，尴尬地停在半空，顿了一下后收回到主人身边，支起王耀微侧的头。

伊万现在知道了他叫王耀——昨夜知道的。直到最后关头，对方竟然还有力气挣脱对他手腕的限制，固执地扭到伊万身边，啃咬他的皮肤，说：“叫我的名字。”

“你叫什么？”伊万在这个空隙里也没忘了下身的动作，狠狠地顶撞过某个点，让对方正松开的口马上又差点化为一阵尖叫。

但他没有，却狠狠抓紧伊万的肩膀，让他不明白这是报复还是其他。他用着尽量压下喘息的声音：“耀，快叫，我是王耀。”

现在不知道是几点，大概太阳还要挺久才会升起，伊万还有机会在这暮色里回忆对方不甚清晰的脸庞。烟头忽明忽暗，大概是醒得更早的缘故，更适应这半黑不暗的光线的王耀发现伊万在盯着自己，尽管可能什么都看不见。他随口问：“也想来一根吗？”

“不，我没抽过。”

王耀在黑暗里笑了。这才是他喜欢的candy，干净，一尘不染，未经世事。好吧，除了昨天晚上自己做的那部分。

“试试吧。”他拿过放在手边的烟盒，倒出一根，用手指夹着送到伊万唇边。对方似乎还有所犹豫，他就用更强硬的动作顶进去，然后在对方还没来得及吐出来的时候欺身上前，用自己嘴里叼着的火星点燃对方的。

然后他们看见，火星在对方的眼睛里出现，又消失。

伊万没有去吸，只是用两唇夹着，眼睁睁地看着王耀把头移开，又一次默默地坐在他身边。接着他试着吸过气，却在烟雾进入喉咙前就吐出来，他实在不想在这样的场景下被呛得咳嗽。

直到王耀把自己口中的烟抽尽，伊万的烟还没下去一半。借着火星的闪烁，王耀看见伊万有些泛红的眼角，又一次轻轻地笑了。然后自然地抽回了伊万的烟，塞进自己两唇之间。

“别勉强了。”他轻缓的语调温柔许多，“再睡一会吧。”

话里似乎有些错误，更累的那个人应该是他。伊万不解，对方却毫不在意地耸耸肩：“我习惯的，睡吧。”

这下伊万睡不着了。他习惯的，这意思是，在自己之前，已经有过许多次了？许多次许多次，他曾在别人面前喘息承欢？握住别人的肩，让别人叫他的名字？

自己好像搞错什么了。

安静的，安静的又度过了一会，王耀大概是以为伊万已经睡着了，便开始起身穿衣。声音尽量放轻，却仍旧偶尔在奇怪的地方停住，然后有一阵细密的叹息声。

穿戴整齐的王耀拉高了自己的领子，在离开前又坐回床边。

“希望还能再见，你确实很可爱，little candy。”
