关于春天的片段

发表于 2014-03-09 更新于 2021-12-02 分类于 Text
本文字数： 1.2k 阅读时长 ≈ 2 分钟

起不来名字。幼驯染提供的梗，和奂说过，没太大意思，可是放不开春风的画面啊。
没有查资料，请不要细究。
其实只是懒得写revolution心情又不是特别好的结果。


“伊万同志！”
北方的城市里，在学院里大声叫出一个异乡的名字也不是什么奇怪的事。俄罗斯族人侧耳倾听背后逐渐密集的脚点声，啪嗒啪嗒像春天雨水催促蘑菇长出来的声音。
“你帮我带点东西到会议室去！”在北方军校里算是小个子的男生说话也像鼓点一样兴奋得好听，每个句子都咚咚咚蹦出来。
“叫我小名我就答应你。”伊万侧侧头靠在肩章上。
“你……！”王耀吸一口气胸口鼓起来像是要骂，看到伊万眨眨眼……又无可奈何地松开。他像一只小松鼠一样狡猾地转头看了看，确认附近没有别人才别开视线咬着下唇哀求：“答应我啦……万尼亚……”
诚实守信是美德。伊万接过王耀递过来的一沓笔记本，用鼻头点点对方正别扭地别开的小鼻子，转身走到花团锦簇的春天里去了。

开完会之后王耀和伊万留在会议室里整理这次的会议记录。伊万托着鼻子看王耀一个人忙活着，在报告上写写抄抄。
春天很安静，他听见王耀比常人都重的吐息声。那声音一定是带着规矩张和的魔力，让他听到王耀的提问时就像上课被教授突然敲醒了一样懵。
王耀说话也没停下笔，“伊万？你也听说旁边航空学院的事了吗？”
伊万知道王耀在指什么。又要到毕业分配的季节了，几乎所有在学校里有所耳闻的情侣都纷纷被派遣到天南海北的地方。他们拉着对方的手哭泣，承诺下下半生的书信。那种东西怎么可能拴住一个人的心？
但命令就是命令。对于军校毕业生来说，不接受命令算什么军人。
“没听说。”伊万撅撅嘴，耸肩表示无辜。
相信自己是他们现在唯一能做的事，伊万必须劝说自己相信自己，他们隐秘的关系即使是最要好的几个同学都不知道。
王耀低垂着眼继续写资料，“嗯，我也没听说多少。”
沙沙沙沙的钢笔声并不那么可爱，王耀终于将工作完成了一半之后抬头叫一声伊万让他帮忙，但是这个偷懒成性的大个子已经缩成了一只熊……缩在桌子上睡着了。
能不能长点心……王耀头疼地去挑挑伊万的白金头发，听见狗熊在梦里反抗了一两声就觉得已经出了气。他想了想，把身上的军大衣褪下来，披到只穿了短外套的伊万身上。
他继续坐着抄写。阳春三月的阳光挑起小风把迎春花从毛玻璃窗外面送进来，落到小狗熊的呼噜声里，点在鬓角边。王耀抄到伊万的名字时下意识抬起头看了看，发现被装点得像春姑娘一样的他还在不满地哼哼。他笑了，悄悄伸出手覆上伊万露在胳膊外的熊爪子。

这样好的风，这样好的景，过了这个年代，就真真不会再有了。
