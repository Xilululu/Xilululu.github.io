2.

“哈……”黎朝阳靠着塑料似的玻璃窗又一次叹气。

其实舱体的速度不快，到现在他也只不过比走在平地上的人高出一些而已。还没有到达会让朝阳害怕的高度，所以即使靠着窗户，或者向脚下的透明地板看，他都不至于害怕，更不会吓得扑到对面的人身上——所以装作害怕扑到Rabiさん身边的计划，似乎实行不了。

他又抱紧了一些怀里的玩偶，把下巴顶到了白熊的脑袋上。不是很习惯佩戴的眼镜夹得他鼻梁发疼，朝阳抬眼，透过平光镜片看向他的队友、他尊敬的人、或许现在可以算作他喜欢的人，嗯……他的恋人，Rabiさん。

他的恋人，光是再想一遍这几个字，就让黎朝阳的耳朵都要红起来。

同时也让他后悔了起来，刚刚上摩天轮的时候，Rabiさん轻轻地在身侧拍了两下——好像理所当然一样，就是理所当然一样，在摩天轮舱这个狭小的空间里，他邀请朝阳坐到自己的身边——而朝阳抱着熊，楞在了舱外。其实也没有多久，顶多一两秒钟吧，却就那么错过了自然而然落座到对方身边的时机。

而且演变到了目前这个面对面坐着一言不发的局面。

Rabi看似并不在意这个，全透明的摩天轮，腾空而起的玻璃房子，他也正凑在窗边向外眺望。而朝阳看着他，恍惚着比较起了Rabi的眼睛和外面的天空哪个更蓝。直到那抹蓝色突然转向自己，Rabi开口：“怎么啦，朝阳，不来看看风景吗？”

黎朝阳摇了摇头，他想看的一直都不是那个。目前占据他脑海的都是自己笨手笨脚地坐到了Rabiさん对面时的样子，Rabi好像惊讶了一小下，然后苦笑起来。

怎么让Rabiさん露出了那样的表情……朝阳咬住嘴唇，把整张脸都脸埋到了白熊的脑袋上。

Rabi的声音又一次响了起来：“看，朝阳，这里看得见整个园区哦。”

“呜……嗯。”朝阳支起头，再消沉下去会让Rabiさん担心的。

“看得见我们过来的地方呢。”Rabi坐得直了些，似乎那样就可以看得更远，“你说，从这里看得见宿舍吗？”

“应该……应该看不见吧。”光路可以逆转，他从来没从别墅看到过这个巨型的造梦工厂，反过来说，这里也看不到那儿。

然后沉默又占据了整舱，刚刚毫无意义的对话让空气变得更尴尬了。朝阳又往窗户边凑近了一点，连呼吸声都有些不敢发出来。他们今天是来干什么的呢，戴起眼镜隐藏身份，穿上差不多的衣服，买不会再戴第二次的头饰与帽子，平时不怎么吃的甜食也是双双两份……普通人、普通人会把这个称作约会，对吧？

摩天轮晃晃悠悠地接近着最高点。

可是自己一直在干什么？乖乖地跟在Rabiさん的后面，被问到想玩什么项目的时候只会说“Rabiさん想玩什么一起玩就好”，因为害羞拒绝着一些恋人之间应该是正常的事。光是想着自己，而最后终于发觉如果只是跟随着Rabiさん，会让后者觉得更难办，于是随便指向了最近的摩天轮。

朝阳从没有像现在一样讨厌自己这样怯弱的性格。

就像现在在摩天轮上，明明应该是最适合恋人的私密的场所，他尽可以坐在Rabi的身边，即使不去说什么让人害羞的话，也可以试着去握住对方的手，感受到对方就在身边。或者说点关于这巨大转轮的传说，那种没有人会去认真地相信、却总是存在的东西：比如……

舱体已经越过了顶峰。

他犹豫地拿捏着怀里白熊软绵绵的两只胳膊，看着地平线一点点向自己接近，准备开口，却突然停下。

那个传说，是什么样的来着？

就像巫婆的诅咒，像传说里最让人害怕的开头：一起坐上摩天轮的恋人，到最后都会分开。

黎朝阳不是迷信的人，可却也被自己难得的好记性吓得慌了神。他揪紧玩偶，感觉不到手上传来的温度。没有缘由、并不会有人信服、轻飘飘的数字传说，可如同今天自己的表现一样，在为这段本来就应该没多少人支持的恋情打上巨大的叉。

他却不想与他分开。

冰冷得要让人发抖的温度里，一个总是让人感到温暖的温度碰触了朝阳。带着茧子的大手轻轻抚摸朝阳的脑袋，如同平日一般，Rabi总能察觉到朝阳那些正在退避的小情绪：“怎么了，朝阳？身体不舒服吗？”

“没、没有……”朝阳摇摇头，被Rabi的声音拉回了一些理智，“我，我在想……今天的我，是不是让Rabiさん感到失望了。”

头顶的抚摸骤停，朝阳一下慌了神，自己是不是又说错了什么？他条件反射般抬起了头，却又在半途就感到后悔，然后撞入另一双眼睛里。

湛蓝眼睛的主人有些许惊讶，却没有任何责怪的情绪，苦笑是有一些吧，但散发着安心的氛围，让朝阳一起平静了下来。Rabi开口：“哈……原来我们想的是一样的事啊。”

“Ra、Rabiさん？”朝阳眨眨眼睛。

Rabi收回手，转移视线的同时捂住了暴露心情的嘴巴：“啊，我也在想，今天是不是有哪里做错了……”他掩住的面上似乎也闪过一点赤色，“……所以朝阳一直都没有看向我呢。”

“诶……”朝阳迅速地在脑中过滤了一遍今天的自己，“诶……不是！不是的……Rabiさん没有做错事情，只是我、是我今天太紧张了，完全什么事都没有做……”他想解释，却又觉得无力起来。

“没有哦，朝阳并不是什么事都没有做。”Rabi回过头来，又一次轻轻地拍了拍朝阳的脑袋，“你出现在这里，就已经是最好的事了。”

黎朝阳突然觉得仓里温暖了许多，温度从抚摸他的那只手传递过来，又好像是从面前这个人身上，一点点扩散到他心里。声音似乎也回到了他的身体里，也许这一次，可以好好地说出来：“Rabiさん……”

而来自地面的声音突然出现，提醒他们快要回到被人包围的地方：“请不要着急，等待船舱和平台等高再开门……”

“啊，快到了啊。”Rabi转头看向门的那一边，身形也开始往那边倾斜。他拿起放在一旁的饮料，准备站起，却被衣角一个小小的力气拉了回来。

拉住他的人也似乎是第一次做出这样的事，朝阳撰住Rabi的衣角，只是这样却好像用完了所有勇气，没有抬起头看他。

“能、能不能，再待一会，Rabiさん……”

于是，摩天轮晃晃悠悠地滑过了最低点，升起时，仓内仍旧坐着那两个人。

“那么，”Rabi坐回了原来的位置，身体前倾，把自己挤进了朝阳的视线里，“你想告诉我什么呢，朝阳？”

“唔！”朝阳被逼得不得不抬头，眼睛先是躲闪了一会儿，然后像是下定决心般闭上，深吸了一口气。再睁开眼睛的时候，他终于能直视起面前的人了，“我、我听过一个摩天轮的传说……”

“什么？”

“就、就是……”朝阳突然停下，像是想起了什么，他抱起自己的白熊，在狭小的舱体里站起，然后，装作自然而无意地，坐到Rabi的身旁。

听起来像是借到了别人的勇气，可是黎朝阳其实还是慌乱得不行。双手交叉在白熊身上，握成拳的掌心被指甲硌得发疼：“就是，就是恋……”

温暖的手掌覆到了他发抖的小拳头上，轻柔地将手指掰开，变成两只手相握的手势。Rabi在他身侧轻轻说：“没事，不用紧张。”

如果是平时，黎朝阳大概会又一次感谢自己身侧有这样一个温柔的人。可是今天，他却突然觉得鼻尖有些酸酸的。他努力地让自己的声音不透露出一丝泪意：“我听说，恋人一起登上摩天轮的话，最后一定会分开……”

握住他的手似乎顿了顿，然后突然抽了出去。朝阳下意识地抬头想知道是不是自己做错了什么，却好像随了Rabi的意，脸颊被捧了起来，不能逃避地被定在了对方面前。

Rabi的笑比起平常又多带了一点狡猾：“我可以认为朝阳是承认我是朝阳的恋人了吗？”

“什……”红潮一下卷上了黎朝阳的脸颊，想要避而不答却又被闭住了退路，他的嘴唇几度张开又闭合，最后眼睛一闭自暴自弃，“我……我们难道不是吗！”

Rabi笑得眼睛都眯了起来，松开手把朝阳揽进怀里，却还在扰乱着他的头发：“是啊是啊，我一直都会在朝阳的身边哦。”

而黎朝阳的脑袋埋在喜欢的人的怀中，红着脸却还是安心地扑入熟悉的气息里。他还是有点放不下那个小诅咒：“那、那关于那个传说……”

“哦，那个啊。那个传说还有下半部分哦，朝阳知道吗？”Rabi松手，让朝阳可以看见自己的脸。

“什么？”

“如果恋人一起乘坐摩天轮便会分开，但是……”Rabi低头，两人的目光齐平。然后，出其不意却理所当然地，行使了他作为恋人的权力。

“如果两人在摩天轮的顶端亲吻的话，便会永远地幸福下去。”

而他们正在离太阳最近的地方。
